`model_example.ipynb` contains a minimal example of running and plotting the model. To generate the figures in the manuscript, run the code in `fig5_model_data_generation.ipynb` to generate the data, and then run the code in `fig5_model_plotting.ipynb` to generate the figures. Use the `model-and-connectome` environment defined in `ENV_stone.yml`. See the readme in the envs folder for installation instructions.
